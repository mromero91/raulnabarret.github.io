Apple anunció su evento el próximo 21 de marzo

El día de ayer una versión beta de Messenger apareció en la app store de Windows

Compartir

Una versión nativa de Messenger está por llegar a Windows 10. El día de ayer una versión beta de esta se encontraba visible desde la app store de Windows. La aplicación funciona para PCs y tablets que corren el más nuevo sistema operativo de Microsoft, pero aún no funciona para dispósitivos móviles. Las funcionalidades y apariencia de la app son muy similares a la versión web en Messenger.com, pero como app nativa cuenta con notificaciones y su propio Live Tile.

Actualmente la versión beta está disponible solamente para un pequeño número de usuarios. « Estamos probando Messenger para Windows 10 con un pequeño porcentaje de personas en algúnos países » —declaró Facebook. También existen rumores que dicen que Facebook de igual manera se encuentra probando una aplicación nativa de Messenger para Mac. Es posible que Facebook anuncie ambas en la próxima conferencia F8 para desarrolladores este abril.
